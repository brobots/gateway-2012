Getting Started with ROBOTC

This guide will help a new user get started using ROBOTC for VEX Cortex.


Step 1: Downloading Master Firmware to VEX Cortex

If you just updated your version of ROBOTC, or are using it with a robot for the first time, you will need to download the Master CPU Firmware and the ROBOTC firmware to your Cortex controller.

 

Ensure that your VEX Cortex is connected to your comptuer via the supplied USB A-to-A cable.

 

Inside of ROBOTC, switch your platform type to VEX Cortex by going to the Robot - Platform Type - Innovation First (IFI) - VEX 2.0 Cortex.

 

Now download the Master Firmware to the VEX Cortex by going back to the "Robot" menu and select "Download Firmware" - "Master CPU Firmware" - "Standard File"

 

The Download Status window will appear and begin the Master Firmware download.

 

When the Download Status window disappears, the Cortex Master Firmware download is complete. You are now ready to download VEXnet Game Controller firmware.

 

Step 2: Downloading ROBOTC Firmware

If you just updated your version of ROBOTC, or are using it with a robot for the first time, you will need to download the ROBOTC User Processor Firmware to your Cortex controller.

 

Ensure that your Cortex is connected to your comptuer via the supplied USB A-to-A cable...


 

Now download the ROBOTC firmware to the VEX Cortex by going back to the "ROBOT" menu and select "Download Firmware" - "ROBOTC Firmware" - "Standard File"

 

The Download Status will appear and begin the ROBOTC firmware download.

 

When the Download Status disappears, the ROBOTC firmware is complete. You are now ready to program with ROBOTC!

 

Step 3 (Optional): Downloading Firmware to VEXnet Joystick

The next step is to install the Master Firmware to the VEXnet Joystick.

 

Ensure that your VEXnet Joystick is connected to your comptuer via the supplied USB A-to-A cable.

 

Now download the VEXnet Joystick Firmware to by going back to the "Robot" menu and select "Download Firmware" - "VEXnet Joystick Firmware" - "Standard File"

 

The Download Status will appear and begin the VEXnet Joystick Firmware download.

 

When the Download Status disappears, the firmware download is complete. You can now sync the Cortex and Joystick by connecting them over the USB A-to-A cable.


Step 4: Opening a Sample Program

ROBOTC includes sample programs to help you get started with learning how to program. To open a sample program, go to the File menu and select Open Sample Program.


Step 5: Getting More Help

ROBOTC includes a useful Help system. You can launch the help system by going to the Help menu and selecting ROBOTC Help, or by pressing the F1 key on your keyboard.

 

Additional help and support can be found at www.robotc.net or by e-mailing support@robotc.net
