Tips and Tricks:

If you just updated your version of ROBOTC, or are using it with a robot for the first time, you will need to download both the IFI Master firmware and the ROBOTC firmware to your robot.

- First download the IFI Master firmware by going to the Robot menu and selecting Download Firmware - Master CPU Firmware

- Then download the ROBOTC firmware by going to the to the Robot menu and selecting Download Firmware - ROBOTC Firmware.

- Finally, make sure you power cycle your VEX (turn it off and then back on) before downloading any code to the robot.

 


ROBOTC includes over 70 sample programs to help you get started with learning how to program. To open a sample program, go to the File menu and select Open Sample Program.

 

ROBOTC also includes built-in templates for creating competition programs. To create a new program using a template, go to the File menu, select New, and then select the type of program you want!
